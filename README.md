# Ivanti Frontend - Temperature Dashboard

### 1. Clone the repository

```bash
git clone https://github.com/andrewwu531/ivanti-frontend.git
cd ivanti-frontend
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the frontend development server

```bash
npm run dev
```

### 4. Run Jest test suite

```bash
npm run test
```
